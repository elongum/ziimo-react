# Ziimo – React App

## Hva er Ziimo?
Barneapp med norske oppdrag. Barn fullfører oppdrag og tjener poeng.
Målgruppe: Barn 4–10 år. Tone: leken, varm, oppmuntrende.

## Design
- Farger: grønn (#82A36C), lys grønn (#B9CDA1), krem (#FDFAD5), hvit, svart
- Font: Baloo 2 (overskrifter), Nunito (brødtekst)
- Border-radius: 12px på kort, 999px på piller/knapper
- Alltid god spacing – padding 16px minimum på kort
- Mobildesign først – alt skal se bra ut på telefon

## CSS-regler
- Bruk CSS variabler fra :root for farger
- Ingen inline styles i produksjonskode
- Hold CSS ryddig – én fil per komponent
- Bruk flexbox for layout, ikke float eller position hacks

## Tech stack
- React med Vite
- Node.js
- Git + GitHub
